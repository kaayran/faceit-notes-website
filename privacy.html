Privacy Policy

Last Updated: October 24, 2025

1. Overview

FACEIT Notes is a browser extension that provides a convenient way to manage player notes on the FACEIT platform. Key features include:

  • Player notes management - add, edit, and delete notes for any player
  • Persistent player tracking - notes linked to FACEIT Player ID
  • Nickname history tracking across username changes
  • Smart note icons showing note status on player cards
  • Search and filter functionality for saved notes
  • Import/Export notes with timestamp-based smart merging
  • Color customization with accessibility presets

This policy explains how the extension processes data while providing these features.

Key Point: This extension does not collect, store, or transmit any personal information to external servers. All data is stored locally in your browser.

2. Data We DO NOT Collect

FACEIT Notes operates with complete privacy in mind. We do NOT collect, store, or transmit:

  • Personally identifiable information (names, email addresses, phone numbers)
  • Browser history or website visits
  • Passwords or login credentials
  • Personal files or documents
  • Location data
  • Usage analytics or tracking data
  • Content of your notes

3. Communication with External Services

The extension interacts with external services solely to provide its core functionality:

3.1. FACEIT Platform

  • Service: www.faceit.com
  • Purpose: The extension operates on FACEIT pages to display note icons and integrate with the platform's user interface
  • Data: No data is sent to FACEIT; the extension only reads publicly available player information

3.2. FACEIT API Proxy

  • Service: faceit-notes-api-proxy.vercel.app
  • Purpose: Proxy service for accessing FACEIT's public API to fetch player information and nickname history
  • Data: Only FACEIT Player IDs are sent to retrieve public player information
  • Note: This proxy does not store or log any requests. It serves solely as a bridge to FACEIT's public API

4. Automatic Technical Information

As with all internet connections, HTTP requests to the proxy server may automatically include standard technical information such as:

  • IP address (required for internet connection)
  • Browser user-agent string
  • Basic request headers
Important: This technical information is not stored, logged, or used for any purpose. The proxy server does not maintain logs or analytics.

5. Local Data Storage

The extension stores data locally in your browser using the browser's built-in storage API. This includes:

  • Player Notes: All notes you create are stored locally on your device, including:
    • Note content
    • Associated FACEIT Player ID
    • Player nicknames (current and historical)
    • Timestamps for note creation and updates
  • Extension Settings: Your preferences including:
    • Icon color customization
    • Extension enabled/disabled state

Important: This data remains on your device and is NEVER transmitted to external servers, except when you explicitly use the export function to save a backup file.

6. Import/Export Functionality

The extension provides import/export features for backing up your notes:

  • Export: Creates a JSON file containing your notes on your local device
  • Import: Reads a JSON file from your local device to restore notes
  • Data Control: You have complete control over these files and where they are stored
  • Privacy: These files are only created and read locally; no data is uploaded to any server

7. Your Rights and Choices

  • Complete Control: You have full control over your data at all times
  • View Data: All your notes are accessible through the extension's interface
  • Delete Data: You can delete individual notes or all data at any time
  • Export Data: Export your notes to a JSON file for backup or migration
  • Removal: Uninstalling the extension will remove all locally stored data
  • No Account Required: The extension works without any account or registration

8. Third-Party Services

This extension interacts with FACEIT's platform and public API. Please review the FACEIT Privacy Policy for information about their data handling practices.

The extension does not integrate with any advertising networks, analytics services, or other third-party tracking services.

9. Children's Privacy

FACEIT Notes does not knowingly collect any information from children. The extension is designed for users of the FACEIT platform, which has its own age requirements. We encourage parents and guardians to monitor their children's internet usage.

10. Changes to This Policy

We may periodically update this privacy policy to reflect changes in the extension's functionality or legal requirements. Any changes will be reflected by updating the "Last Updated" date at the top of the page.

Significant changes will be announced through:

  • Extension update notes
  • GitHub repository announcements
  • This privacy policy page

Continued use of the extension after changes indicates acceptance of the updated policy.

11. Legal Basis (for EU Users)

For users in the European Union, our legal basis for data processing is:

  • Legitimate Interest: Processing is necessary to provide the extension's functionality
  • Consent: By installing and using the extension, you consent to the data processing described in this policy
  • Data Minimization: We process only the minimum data necessary to provide functionality

As all data is stored locally on your device and not transmitted to our servers, GDPR data subject rights are maintained through your direct control over the data.

12. Data Security

We implement appropriate technical measures to ensure the security of the extension:

  • Data is stored using browser's secure storage API
  • No transmission of sensitive data to external servers
  • Extension operates within browser's security sandbox
  • Regular security updates and code reviews
  • Open-source code available for community security audits

13. Open Source Transparency

  • Report security concerns through GitHub issues
